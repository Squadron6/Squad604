Setup
-----

### To install/run Fish ###
1. Install autoconf and the g++ compiler
2. Install curses:
- Debian or Ubuntu:
```
sudo apt-get install libncurses5-dev
```
- RedHat, CentOS, or Amazon EC2:
```
sudo yum install ncurses-devel
```
### To run Pisces ###
1. Install g++ compiler
2. Install Python
3. Install clang
4. Go into the Pisces directory
5. Run make
6. Run `./Pisces`
